# Computer Sim Projects

Simulation projects completed for **CSCI-UA 330: Introduction to Computer Simulation**  
**Professor:** Charles Peskin

---

## Projects

### Asteroid Deflection Simulation
**Minimum Energy Impulse to Barely Miss Earth**

This project studies asteroid trajectory deflection and computes the minimum-energy impulse required for an asteroid to just miss Earth.

---

### Cordyceps Colony Simulation
**Agent-Based Simulation of Cordyceps Infection in an Ant Colony**

This project models the spread of a Cordyceps-like infection through an ant colony using an agent-based stochastic simulation.

**Collaborators:**  
- [Hugh Richards](https://github.com/xHJRM)
- [Damiane Kapanadze](https://github.com/DamianeKapanadze)

---

### Simulated Computer Arithmetic
**Building Arithmetic Routines on a Simulated Computer**

This project builds arithmetic routines on a minimal simulated computer, including operations such as multiplication, division, and GCD, to explore low-level computation and control flow.

**Collaborators:**  
- [Hugh Richards](https://github.com/xHJRM)
- [Damiane Kapanadze](https://github.com/DamianeKapanadze)

---

## About

These three projects were completed over the course of the semester and reflect different applications of computer simulation:
- orbital mechanics,
- biological agent-based modeling,
- and low-level computational architecture.
